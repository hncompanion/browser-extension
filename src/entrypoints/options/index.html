Prompt customization

Override the built-in system and user prompts used when your provider generates a summary. Most users never need to change these.

Keep this off to use the built-in system and user prompts. Affects only summaries your provider generates — not cached server summaries.

The user prompt uses JavaScript template literals. Include these variables:

  1. ${title} — the post title.
  2. ${text} — the formatted comment text.

Include both ${title} and ${text} for best results.

Requires ${title} and ${text} when enabled.

Changes are applied after you press Save and stored with browser sync.
