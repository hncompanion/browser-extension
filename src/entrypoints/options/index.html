How this works:

ðŸ’¡ The HN Companion server periodically fetches stories from the Hacker News homepage and creates LLM summaries which are cached on our servers.

When this option is enabled, and you request a summary for a story that's already cached, the extension will display it immediately without needing to use your own API keys.

If you want a different summary than the cached one, you can generate a fresh summary at any time. Just use the link in the header of the summary panel - this will use your selected AI provider and your own API key.

Free Options

ðŸ’¡ Free local summarization using Ollama

For this extension to work with Ollama:

  1. Make sure Ollama is running to see your downloaded models in the dropdown
  2. Enable CORS for the extension:

    On Mac (run on terminal):

    launchctl setenv OLLAMA_ORIGINS "https://news.ycombinator.com,chrome-extension://*,moz-extension://*"

    On Windows:

    setx OLLAMA_ORIGINS "chrome-extension://*,https://news.ycombinator.com"
  3. Restart Ollama for the CORS settings to take effect

Note: The model dropdown will automatically show your locally downloaded models when Ollama is running properly. Make sure the settings persist across machine restarts.
